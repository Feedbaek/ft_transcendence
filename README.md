## 구현 기능

### 기능 1
42OAuth 인증을 사용한 로그인, 2차 인증 기능과 유저 프로필

### 기능 2
실시간 채팅과 친구목록을 포함한 다양한 상호작용 기능

### 기능 3
ping-pong 게임을 모방하여 만든 실시간 게임.
혼자하기, 랭크전, 친선전 3가지 모드 존재

<br>
